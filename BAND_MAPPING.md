# Форматы файлов для расчёта индексов

## Важная информация о каналах

Для корректного расчёта индексов необходимо правильно понимать структуру многоканальных TIFF файлов.

### Sentinel-2 (13 каналов)

| Номер канала | Название | Длина волны | Используется для |
|-------------|----------|-------------|------------------|
| B01 | Coastal Aerosol | 443 nm | - |
| B02 | Blue | 490 nm | EVI |
| B03 | Green | 560 nm | NDWI, GNDVI |
| B04 | Red | 665 nm | NDVI, EVI, NBR, GEMI |
| B05 | Red Edge 1 | 705 nm | - |
| B06 | Red Edge 2 | 740 nm | - |
| B07 | Red Edge 3 | 783 nm | - |
| B08 | NIR | 842 nm | NDVI, NDWI, EVI, NBR, GNDVI, GEMI |
| B8A | NIR Narrow | 865 nm | - |
| B09 | Water Vapour | 945 nm | - |
| B10 | Cirrus | 1375 nm | - |
| B11 | SWIR 1 | 1610 nm | NBR |
| B12 | SWIR 2 | 2190 nm | - |

### Варианты загрузки файлов

#### 1. Готовый многоканальный файл

Если у вас уже есть многоканальный TIFF файл Sentinel-2:

- **Каналы должны идти в порядке B01, B02, B03, ... B12**
- Канал 1 в файле = B01
- Канал 2 в файле = B02
- Канал 3 в файле = B03
- Канал 4 в файле = B04 (Red)
- ...
- Канал 8 в файле = B08 (NIR)

**Важно!** Если ваш файл имеет другую структуру, используйте "Объединение каналов".

#### 2. Объединение отдельных каналов

Рекомендуемый способ! Вы загружаете отдельные файлы каналов (B01.tif, B02.tif, и т.д.) и система:

1. Автоматически определяет канал по названию файла (B01, B02, ...)
2. Сортирует их в правильном порядке
3. Объединяет в один файл с каналами в нужном порядке

### Индексы и требуемые каналы

| Индекс | Каналы Sentinel-2 | Формула |
|--------|-------------------|---------|
| NDVI | B04 (Red), B08 (NIR) | (NIR - Red) / (NIR + Red) |
| NDWI | B03 (Green), B08 (NIR) | (Green - NIR) / (Green + NIR) |
| EVI | B02 (Blue), B04 (Red), B08 (NIR) | 2.5 × (NIR - Red) / (NIR + 6×Red - 7.5×Blue + 1) |
| NBR | B04 (Red), B08 (NIR), B11 (SWIR) | (NIR - SWIR) / (NIR + SWIR) |
| GNDVI | B03 (Green), B08 (NIR) | (NIR - Green) / (NIR + Green) |
| GEMI | B04 (Red), B08 (NIR) | Сложная нелинейная формула |

### Как работает выбор каналов

**В коде используется маппинг:**

```python
satellite_bands = {
    "Sentinel-2": {
        "Blue": 2,   # B02
        "Green": 3,  # B03
        "Red": 4,    # B04
        "NIR": 8,    # B08
        "SWIR": 11,  # B11
        # ...
    }
}
```

Значения — это **номера каналов** (начиная с 1) в многоканальном файле, которые соответствуют реальным спектральным каналам Sentinel-2.

### Пример расчёта NDVI

1. Пользователь загружает многоканальный файл
2. Система проверяет: для NDVI нужны Red (B04) и NIR (B08)
3. Из маппинга: Red = канал 4, NIR = канал 8
4. Преобразуем в индексы массива (0-based): Red → индекс 3, NIR → индекс 7
5. Читаем 4-й и 8-й каналы из файла
6. Рассчитываем: (NIR - Red) / (NIR + Red)

## Рекомендации

1. **Используйте "Объединение каналов"** — это гарантирует правильный порядок
2. **Проверяйте названия файлов** — система автоматически определит B01, B02, и т.д.
3. **Убедитесь, что есть все нужные каналы** — для каждого индекса нужны свои каналы
